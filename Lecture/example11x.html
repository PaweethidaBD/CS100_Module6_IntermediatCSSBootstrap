Bootstrap Float Utilities

This paragraph has an image floated to the left using Bootstrap's .float-start. Text wraps automatically around the image. The .me-3 utility adds right margin, and .mb-3 adds bottom margin to keep spacing when the text wraps on small screens. You can use this approach for small layout tweaks, like placing profile pictures in articles or floating icons in a block of text.
